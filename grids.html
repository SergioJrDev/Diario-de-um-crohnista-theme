Grid

Sistema de grid baseado em flexbox, com algumas classes simples, como no Bootstrap.

Para criar uma grid, defina a classe .column e espefique o tamanho dos elementos dentro deles. 


<div class="column">
	<div class="sm-6-6"></div>
	<div class="sm-3-6"></div>
	<div class="sm-3-6"></div>
</div>

Toda mágica acontece pela função do Stylus 


grid(x) {
	for n in (1..x) {
	  .sm-{n}-{x} {
		padding: 0 15px;
		width ((0% + n / x * 100));
	  }
	}
}
//Chamando a função e passando a quantidade de colunas
grid(12);
